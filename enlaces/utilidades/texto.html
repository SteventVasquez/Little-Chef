Texto

El texto es el ingrediente principal de cualquier interfaz. Con estas clases, puedes controlar el tamaño, el peso., alineacion y color de tus textos de forma rápida.

Tamaños de fuente

Utiliza fs- (font-size) para cambiar el tamaño del texto. El sistema sigue una jerarquía donde el número menor es el mas grande.

Alineación del texto

Ubica el texto exactamente donde lo necesites dentro de su contenedor.

  • .text-start : Alinea el texto a la izquierda (por defecto).
  • .text-center : Centra el texto, muy útil para tarjetas.
  • .text-end : Alinea el texto a la derecha.

    • Cualquiera puede cocinar

    Cualquiera puede cocinar

    Cualquiera puede cocinar

    <p class="fs-4 text-start m-2">Cualquiera puede cocinar</p>
<p class="fs-4 text-center m-2">Cualquiera puede cocinar</p>
<p class="fs-4 text-end m-2">Cualquiera puede cocinar</p>

    Transformación del texto

    Estas utilidades te permiten cambiar rápidamente entre mayúsculas y minúsculas para dar jerarquía a tus textos.

    manten tu estación limpia

    MANTEN TU ESTACIÓN LIMPIA

<p class="text-lowercase fs-5 m-2">MANTEN TU ESTACIÓN LIMPIA</p>
<p class="text-capitalize fs-5 m-2">manten tu estación limpia</p>
