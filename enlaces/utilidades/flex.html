Organización Flexible (Flexbox)

Flexbox es nuestra herramienta para alinear y distribuir elementos de forma automática. Es como tener un chef de sala que organiza cada plato para que siempre se vea perfecto, sin importar el tamaño de la mesa.

El Contenedor (.d-flex)

Para que cualquier regla de organización funcione, el padre debe ser un contenedor flexible. Es el tablero de cocina donde ocurre la magia.

Direccion y flujo

Decide cómo se ordenan los elementos dentro del contenedor.

  • .flex-row: Los elementos se alinean como cubiertos en una fila (horizontal).
  • .flex-column: Los elementos se apilan como una torre de panqueques (vertical).
  • .flex-wrap: Si no hay espacio, los elementos pasan a la siguiente fila (como galletas en una bandeja).

Alineación de los "Platos"

Controla el espacio y la posición de tus elementos con precisión:

Clase Función
justify-content-* Controla el alineamiento horizontal.
align-items-* Controla el alineamiento vertical de todos los elementos a la vez.
align-self-* Permite que un solo elemento se rebele y se alinee diferente al resto.
