Visibilidad (Display)

Estas clases sirven para mostrar u ocultar elementos. Son especialmente útiles para adaptar tu diseño a celulares, tablets y computadoras sin cambiar el HTML.

  • .d-none: Oculta el elemento por completo (desaparece del plato).
  • .d-block: Muestra el elemento como un bloque (ocupa todo el ancho disponible).

Control responsivo

Puedes combinar estas clses con los tamaños de pantalla (sm,md,lg,etc.)

  • .d-{tamaño}-none: Oculta el elemento a partir de ese tamaño de pantalla.
  • .d-{tamaño}-block: Muestra el elemento a partir de ese tamaño de pantalla.
