Espaciado y Distribución

En la cocina, la presentación lo es todo. Nuestras clases de espaciado te permiten controlar la distancia entre ingredientes (elementos) para que tu diseño tenga mejor aspecto visual.

Margins y Paddings

Los paddings y margins predefinidos usan una medida estándar basada en 1rem de donde tenemos las demás medidas.

  • 0-0rem
  • 1-0.25rem
  • 2-0.50rem
  • 3-1rem
  • 4-1.5rem
  • 5-3rem

Notación utilizada

A continuación te explicamos la notación con la que se trabajo para los nombres de clase respectivas a margins y paddings

  • m - es la notación para margin
  • p - es la notación para padding

Donde los lados son:

  • t - equivalente a colocar top para clases que establecen margin-top o padding-top
  • b - equivalente a colocar bottom para clases que establecen margin-bottom o padding-bottom
  • l - equivalente a colocar left para clases que establecen margin-left o padding-left
  • r - equivalente a colocar right para clases que establecen margin-right o padding-right
  • x - equivalente a colocar right y left aplica el espaciado en el eje horizontal del elemento.
  • y - equivalente a colocar top y bottom aplica el espaciado en el eje vertical del elemento.
$spacer: 1rem;
.mt-0 {
  margin-top: 0;
}

.ml-1 {
  margin-left: ($spacer * .25);
}
           
.px-2 {
  padding-left: ($spacer * .5);
  padding-right: ($spacer * .5);
}

.p-3 {
  padding: $spacer;
}

Espaciado entre elementos (Gap)

La propiedad gap define el espacio (la separación) entre los elementos hijos de un contenedor flexible o de rejilla, sin añadir espacio en los bordes exteriores del contenedor.

  • .gap-{}: Aplica un espacio uniforme tanto horizontal como verticalmente entre los elementos hijos.
  • A diferencia de los márgenes, el gap solo se aplica entre los elementos, lo que facilita el alineamiento perfecto con los bordes del contenedor padre.
  • Para que esta clase te funcione, el contenedor debe tener una propiedad de diseño activa, como display: flex.
