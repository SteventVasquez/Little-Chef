Badges (Insignias)

Pequeños indicadores para resaltar estados, categorías o notificaciones en tus platillos.

"El detalle final en la presentación. Usa insignias para destacar lo que realmente importa."

Badges Temáticas

Utiliza las clases utilitarias de fondo (.bg-remy, .bg-ego) combinadas con el color de texto claro (.text-light) para darle vida a tus insignias.

Remy Ego Collete Linguini 
<span class="badge bg-remy text-light px-3 py-1 rounded">Remy</span>
<span class="badge bg-ego text-light px-3 py-1 rounded">Ego</span>
<span class="badge bg-colette text-light px-3 py-1 rounded">collete</span>
<span class="badge bg-linguini text-light px-3 py-1 rounded">Linguini</span>

Tamaño Automático

Las insignias heredan el tamaño de la tipografía del elemento padre. Colócalas dentro de encabezados para que se escalen correctamente.

Sopa de Cebolla Nuevo

Pato a la Naranja Especial

Crème Brûlée Postre
<h1 class="font-remy">Sopa de Cebolla <span class="badge bg-remy text-light rounded">Nuevo</span></h1>
<h3 class="font-remy">Pato a la Naranja <span class="badge bg-ego text-light rounded">Especial</span></h3>
<h5 class="font-remy">Crème Brûlée <span class="badge bg-colette text-light rounded">Postre</span></h5>
