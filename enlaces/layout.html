Contenedores

El plato donde comienza cada creación. El contenedor define el espacio y los límites de tu banquete visual. Los contenedores son el componente fundamental de Little Chef para establecer el ancho máximo del contenido y centrarlo dentro del viewport.

Ofrecemos dos variantes principales según la necesidad de tu "receta" de diseño:

  • .container: Un contenedor responsivo con anchos máximos fijos que saltan en cada breakpoint (móvil, tablet, escritorio)
  • .container-fluid: Un contenedor de ancho completo que abarca el 100% de la pantalla en todo momento, ideal para layouts fluidos.

La tabla ilustra cómo se comporta el ancho máximo de .container y .container-fluid

Sistema de Rejilla

El sistema de rejilla de Little Chef utiliza un modelo de 12 columnas basado en Flexbox para crear diseños responsivos.

Para que el grid funcione, debes respetar la jeraquía de componentes:

  • Contenedor (.container): Define el área de trabajo y centra el contenido
  • Fila (.row): Actúa como el envoltorio para las columnas y elimina los márgenes laterales mediante márgenes negativos.
  • Columnas (.col): Hay 12 columnas de plantilla disponibles por fila, lo que permite crear diferentes combinaciones de elementos que abarcan cualquier número de columnas.

Clase .row

La clase .row es el contenedor que agrupa las columnas. Utiliza display:flex y flex-wrap para organizar las columnas en una fila.

Columnas ancho fijo

La clase .col-{n} (donde n es un número del 1 al 12) permite definir el ancho exacto que una columna ocupará en la fila.

Col-4
Col-8
<div class="container text-center">
    <div class="row">
        <div class="col-4 border">Col-4</div>
        <div class="col-8 bg-info border border-start-0">Col-8</div>
    </div>
</div>

Columnas responsivas

Puesdes usar clases .col-sm-*, .col-md-*, hasta .col-xxl-* para definir tamaños según el ancho de la pantalla.

Columna 2

Las clases .align-self-* permiten alinear individualmente cada columna dentro de una fila.
