Little Chef

"Cualquiera puede codificar"

Un framework CSS ligero, modular y altamente personalizable inspirado en la alta cocina parisina. Diseñado para que construyas interfaces con la misma pasión y orden con la que un chef crea sus platillos.

Ver Documentación Ver en GitHub

Los Ingredientes de Nuestro Framework

Componentes Modulares

Botones, tarjetas, alertas y formularios listos para servirse directamente en tu HTML sin escribir CSS extra.

Temas de Personajes

Clases utilitarias con paletas de colores exclusivas basadas en la esencia de Remy, Ego y Collete.

Responsivo

Un sistema de grillas (Grid) flexible que adapta tus diseños a cualquier pantalla, desde móviles hasta monitores de escritorio.
